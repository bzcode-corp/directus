# Composite PK Bundle

Пакет расширений Directus, добавляющий поддержку таблиц с составными первичными ключами.

## Возможности

- API endpoint `/citems` с CRUD для коллекций с составными PK;
- App‑модуль **Composite Content** для просмотра и редактирования таких коллекций в Data Studio.

Расширение активируется флагом `COMPOSITE_PK_ENABLED=true`. При выключенном флаге новые маршруты и модуль не регистрируются.

## Сборка

```bash
pnpm install
pnpm build
```

Собранные файлы появляются в `dist/`.

## Установка

Скопируйте содержимое `dist/` в каталог расширений Directus, по умолчанию `/directus/extensions` (либо путь из переменной окружения `EXTENSIONS_PATH`).

## Примечание

Интроспекция составных ключей реализована для PostgreSQL.
